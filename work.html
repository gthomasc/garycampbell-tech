Headline initiative: 4-Year IT Modernization Roadmap

Built and stewarding a phased multi-year roadmap designed around a deliberate sequencing model — Stabilize before Harden, Harden before Adopt, Adopt before Modernize — to avoid the common mistake of treating all four as parallel work streams.

  • •Stabilize: Server replacement, AD cleanup, backup audit (DATTO), MFA expansion, addressing aging infrastructure and EOL equipment
  • •Harden: Security gap closure, conditional access, vulnerability management, vendor security review, PCI-DSS readiness
  • •Adopt: Microsoft 365 collaboration adoption (Teams, Planner, Copilot), AI workflow integration with director buy-in
  • •Modernize: Aruba wireless refresh and Aruba Central migration, POS refresh, surveillance upgrades, retail technology modernization
